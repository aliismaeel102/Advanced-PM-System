# Integrated Project Management Decision Support System (IPMDSS)

**Developed by:** Ali Ismail Ali Bayoumi Mohamed  
**Academic Department:** Information Systems (IS)  

## Overview
This advanced tool is designed to assist project managers in the initial phases of project planning. It integrates risk management, financial forecasting, and resource budgeting into a single analytical framework.

## Key Modules
1. **Risk Registry Matrix:** Uses a probability-impact matrix to classify risks into Critical, High, Medium, or Low.
2. **Financial Forecasting:** Implements NPV (Net Present Value) and ROI (Return on Investment) calculations to evaluate economic feasibility.
3. **Resource Budgeting:** Allows for detailed allocation of funds across different project departments.
4. **Automated Reporting:** Generates a professional `.txt` report summarizing all inputs and calculated results.

## Technical Details
- **Language:** Python 3.x
- **Inputs:** Dynamic user data for costs, risks, and timelines.
- **Output:** On-screen analysis and a permanent report file.

## Usage
Run `main.py` and follow the command-line prompts to input your project details.
